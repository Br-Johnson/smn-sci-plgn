# SMN Source Notes

- Canonical namespace root: `https://w3id.org/smn`
- Published JSON-LD surface: `https://salmon-data-mobilization.github.io/salmon-domain-ontology/smn.jsonld`
- The shared ontology is the preferred first stop for cross-organization, policy-neutral terms.
- If a needed term is clearly DFO-specific or profile-scoped, check `gcdfo` next rather than forcing it into `smn`.
- This skill is a lookup surface, not a full semantic crosswalk engine.
